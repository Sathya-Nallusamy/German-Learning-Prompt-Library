## ğŸ”„ Variations

### Variation 1: Announcement Mode
Add this to the prompt:
```
Instead of a dialogue, create a PUBLIC ANNOUNCEMENT (train station, airport, store). Make it realistic with background noise descriptions. Test if I caught the key information.
```

### Variation 2: Speed Challenge
Add this to the prompt:
```
First show me the dialogue in "slow" form with full sentences. Then show a "fast/natural" version with contractions and casual speech. Test me on both.
```

### Variation 3: Numbers Focus
Add this to the prompt:
```
Include many numbers in the dialogue (times, prices, phone numbers, addresses). Focus the comprehension questions on whether I caught the numbers correctly.
```

### Variation 4: Repeat & Expand
Add this to the prompt:
```
After the first exercise, create a SECOND dialogue with the same characters continuing their conversation. This helps me follow an ongoing story.
```

## ğŸ“š Listening Strategies

### Before "Listening" (Pre-reading)
- Read the situation description
- Review key vocabulary
- Know what information to find

### During "Listening" (First read)
- Don't stop at unknown words
- Focus on the main idea
- Note numbers, names, times

### After "Listening" (Comprehension)
- Answer from memory first
- Check the transcript only after answering
- Note new expressions you learned
