## ğŸ”„ Variations

### Variation 1: Review Mode
Add this to the prompt:
```
Before teaching new words, first quiz me on these words I learned yesterday: [list your previous words]
```

### Variation 2: Harder Challenge
Add this to the prompt:
```
Make the example sentences slightly more complex and add one sentence where I need to identify the correct article (der/die/das).
```

### Variation 3: Conversation Focus
Add this to the prompt:
```
After the quiz, create a mini-dialogue (4-6 lines) using at least 5 of these words that I might use in real life.
```

### Variation 4: Writing Practice
Add this to the prompt:
```
After the quiz, ask me to write 3 of my own sentences using the new vocabulary. Then correct them.
```
